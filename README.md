# Vrinda Store Sales Analysis (Excel Dashboard)

## Project Overview
This project analyzes Vrinda Store sales data using Microsoft Excel to identify sales trends, customer behavior, and business performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Dashboard Features
- Total Sales
- Total Orders
- Top 5 Products
- Monthly Sales Trend
- Sales by Category
- Sales by State
- Sales by Gender

## Key Insights
- Identified top-performing products.
- Compared monthly sales performance.
- Analyzed customer demographics.
- Created an interactive dashboard using slicers.
